## FinchB4G

Basic website frontend built with Next.js, TypeScript, and the App Router. It is set up as a simple landing page starter that you can customize first and connect to Vercel later.

## Run locally

Start the development server:

```bash
npm run dev
```

Then open `http://localhost:3000`.

## Main files

- `app/page.tsx`: homepage content and section structure
- `app/page.module.css`: homepage styling
- `app/layout.tsx`: site metadata and font setup
- `app/globals.css`: global design tokens and base styles

## Deploy to Vercel later

When you are ready:

1. Push this repo to GitHub.
2. Import the repo into Vercel.
3. Accept the default Next.js build settings.
4. Add your custom domain after the first successful deploy.

## Customize next

- Replace the placeholder marketing copy in `app/page.tsx`.
- Add your own images or brand assets.
- Expand the app with new routes, forms, or API integrations as needed.
